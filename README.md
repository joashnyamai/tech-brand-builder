# Malila Nyamai - Portfolio

## Overview

Welcome to the professional portfolio website of **Malila Nyamai**, a dedicated **Full-Stack Engineer**, **QA Specialist**, and **IT Consultant**. This project showcases my professional journey, technical expertise, and the impactful projects I've built.

Built with modern web technologies, this portfolio is designed to be responsive, accessible, and visually engaging, reflecting my commitment to high-quality software development.

## Features

- **Dynamic Hero Section**: An engaging introduction with a professional overview.
- **Detailed About Me**: A comprehensive look into my professional background and philosophy.
- **Experience Timeline**: An interactive timeline highlighting my career milestones and roles.
- **Project Showcase**: A detailed gallery of my key projects, complete with descriptions and tech stacks.
- **Technical Skills**: A categorized overview of my proficiency in various languages and tools.
- **Certifications**: A section dedicated to my professional certifications and achievements.
- **Community Involvement**: Highlighting my contributions and engagements within the tech community.
- **Impact & Mentorship**: Showcasing my role in mentoring and driving positive change in the industry.
- **Contact Form**: An integrated form for potential collaborations and inquiries.
- **Responsive Layout**: Fully optimized for a seamless experience across desktop, tablet, and mobile devices.

## Tech Stack

This project is built using the following technologies:

- **[Vite](https://vitejs.dev/)**: Next Generation Frontend Tooling for fast development and building.
- **[React](https://reactjs.org/)**: A JavaScript library for building user interfaces.
- **[TypeScript](https://www.typescriptlang.org/)**: Typed JavaScript for safer and more scalable code.
- **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework for rapid UI development.
- **[shadcn/ui](https://ui.shadcn.com/)**: Re-usable components built using Radix UI and Tailwind CSS.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Ensure you have the following installed on your system:
- **Node.js** (version 16.0 or higher recommended)
- **npm** (usually comes with Node.js)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    cd tech-brand-builder
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

### Running the Application

To start the development server:

```bash
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) (or the port shown in your terminal) to view it in the browser.

### Building for Production

To create a production-ready build:

```bash
npm run build
```
