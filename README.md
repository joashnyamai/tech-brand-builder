# Malila Nyamai - Portfolio

## Project info
This is the portfolio website for Malila Nyamai, a Full-Stack Engineer, QA Specialist, and IT Consultant.

## Technologies
This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
